# yup-nl

```
NAME:
   nl - number lines of files

USAGE:
   nl [OPTIONS] [FILE...]

   Write each FILE to standard output, with line numbers added.
   With no FILE, or when FILE is -, read standard input.

VERSION:
   dev

GLOBAL OPTIONS:
   --body-numbering string, -b string    use STYLE for numbering body lines (a=all, t=non-empty, n=none)
   --number-separator string, -s string  add STRING after (possible) line number
   --starting-line-number int, -v int    first line number for each section (default: 0)
   --line-increment int, -i int          line number increment at each line (default: 0)
   --number-width int, -w int            use NUMBER columns for line numbers (default: 0)
   --number-format string, -n string     insert line numbers according to FORMAT (ln, rn, rz)
   --help, -h                            show help
   --version                             print version information and exit
```
